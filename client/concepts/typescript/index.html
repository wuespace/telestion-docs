TypeScript

The Telestion Client Ecosystem uses TypeScript for both its libraries (adding type support from the get-go) and also (by default) the client code.

TypeScript is, by their own words, "JavaScript with syntax for types" and, thus, makes Telestion Clients strongly typed while still retaining the benefits of JavaScript.

Since the usage of TypeScript makes developing Clients type-safe, you get much better IDE integration like autocompletion. For example, you can use Visual Studio Code (which is free) or WebStorm which both have excellent TypeScript support.
